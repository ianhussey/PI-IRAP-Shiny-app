Calculate PIIRAP scores for the IRAP

Purpose

This Shiny app can be used to calculate PIIRAP for data generated by the Implicit Relational Assessment Procedure. The PIIRAP is an alternative scoring algorithm to the D score (Greenwald et al., 2003), based on a more robust effect size measure, the PI (Thas et al., 2012). For an article length discussion of the PIIRAP, see (De Schryver, Hussey, De Neve, Cartwright, & Barnes-Holmes (under review: see OSF page for pre print).

Please note that this app is for scoring data, but not for cleaning it. The raw data produced by a given implementation of the IRAP must be processed into the necessary format before it can be scored. Note that some implementations of the IRAP may be more ameniable to this processing, depending on the format of its output.

Data Formatting

The data must be uploaded as a .csv file (sep=";", dec=","). Variables names should be included on the first row, and must include the following:

  • ID - the participant identifier
  • PR - the block pair number
  • TN - the trial number
  • CR - the block within the block pair (i.e., consistent or inconsistent)
  • CO - accuracy (either 0 or 1)
  • LA - reaction time
  • TT - trial type
